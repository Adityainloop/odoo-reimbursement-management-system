# Odoo Reimbursement Management System

## 🚀 Problem Statement
Expense reimbursement system with multi-level approvals and smart rules.

## 🧠 Features
- Employee expense submission
- Multi-level approval workflow
- Conditional approval logic
- Currency conversion
- OCR receipt scanning (optional)

## 🛠 Tech Stack
- Backend: Python (FastAPI)
- Frontend: React / HTML-CSS-JS
- Database: PostgreSQL
